# Create a project folder for iphra analysis

Create a project folder for iphra analysis

## Usage

``` r
create_iphra_analysis_template(folder_path, ...)
```

## Arguments

- folder_path:

  Path for the new folder

- ...:

  Extra information collected from the RStudio wizard

## Value

Folder with a copy of the iphra_analysis

## Examples

``` r
if (FALSE) { # \dontrun{
create_iphra_analysis_template("path/to/folder")
} # }
```
