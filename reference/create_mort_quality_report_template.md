# Create a project folder for mort quality report

Create a project folder for mort quality report

## Usage

``` r
create_mort_quality_report_template(folder_path, ...)
```

## Arguments

- folder_path:

  Path for the new folder

- ...:

  Extra information collected from the RStudio wizard

## Value

Folder with a copy of the mort_quality_report

## Examples

``` r
if (FALSE) { # \dontrun{
create_mort_quality_report_template("path/to/folder")
} # }
```
