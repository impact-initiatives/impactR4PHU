# Reformat Mortality Data

Reformat Mortality Data

## Usage

``` r
reformat_mortality(df)
```

## Arguments

- df:

  long dataframe containing (roster/left/died dfs)

## Value

dataframe with recoded values for: -sex -date_dc -date_recall -date_join
-date_birth -date_left -date_death -join -left -birth -death
-cause_death -location_death

``` r

```
