# Create a project folder for iycf cleaning

Create a project folder for iycf cleaning

## Usage

``` r
create_iycf_cleaning_template(folder_path, ...)
```

## Arguments

- folder_path:

  Path for the new folder

- ...:

  Extra information collected from the RStudio wizard

## Value

Folder with a copy of the iycf_cleaning

## Examples

``` r
if (FALSE) { # \dontrun{
create_iycf_cleaning_template("path/to/folder")
} # }
```
