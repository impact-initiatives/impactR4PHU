# Create a project folder for iycf analysis

Create a project folder for iycf analysis

## Usage

``` r
create_iycf_descriptive_analysis_template(folder_path, ...)
```

## Arguments

- folder_path:

  Path for the new folder

- ...:

  Extra information collected from the RStudio wizard

## Value

Folder with a copy of the iycf_analysis

## Examples

``` r
if (FALSE) { # \dontrun{
create_iycf_descriptive_analysis_template("path/to/folder")
} # }
```
