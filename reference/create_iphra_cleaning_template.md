# Create a project folder for iphra cleaning

Create a project folder for iphra cleaning

## Usage

``` r
create_iphra_cleaning_template(folder_path, ...)
```

## Arguments

- folder_path:

  Path for the new folder

- ...:

  Extra information collected from the RStudio wizard

## Value

Folder with a copy of the iphra_cleaning

## Examples

``` r
if (FALSE) { # \dontrun{
create_iphra_cleaning_template("path/to/folder")
} # }
```
